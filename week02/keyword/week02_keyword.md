# 📋 2주차 KEYWORD 📋

## 👩🏻‍💻 Activity
Activity는 안드로이드 앱의 전체 화면 단위이며, 사용자와의 주요 상호작용이 시작되는 진입점이다.

<br>

## 👩🏻‍💻 Fragment
Fragment는 Activity 내부에서 재사용 가능한 부분 화면이며, 하나의 Activity 안에서 여러 화면을 효율적으로 구성할 수 있게 해준다.

<br>

## 👩🏻‍💻 FragmentContainerView
FragmentContainerView는 Fragment가 동적으로 들어가는 컨테이너 역할을 한다.

<br>

## 👩🏻‍💻 BottomNavigationView
BottomNavigationView는 하단 탭 UI를 제공하며, 각 탭과 Fragment를 연결해 화면 전환을 구현할 수 있다.

<br>

## 👩🏻‍💻 Intent
Intent는 안드로이드 시스템에 특정 작업을 요청하는 메시지 객체이며, Activity 간 이동과 데이터 전달에 사용된다.

<br>

## 👩🏻‍💻 NavGraph
NavGraph는 Fragment 간 이동 경로를 XML로 정의하는 Navigation Component의 핵심 구성 요소이다.

<br>

## 👩🏻‍💻 Safe Args
Safe Args는 Navigation에서 Fragment 간 데이터를 타입 안전하게 전달하기 위해 자동 생성 코드를 제공하는 도구이다.

<br>

## 👩🏻‍💻 ViewBinding
ViewBinding은 XML 뷰를 타입 안전하게 참조할 수 있도록 도와주는 기능이며, findViewById를 대체할 수 있다.

<br>
