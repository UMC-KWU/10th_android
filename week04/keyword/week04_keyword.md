# 📋 4주차 KEYWORD 📋

## 👩🏻‍💻 Main Thread
안드로이드에서 UI 작업을 담당하는 가장 중요한 실행 흐름

<br>

## 👩🏻‍💻 ANR(Application Not Responding)
메인 스레드가 오래 막혀 앱이 응답하지 않는 상태

<br>

## 👩🏻‍💻 동기(Synchronous)
한 작업이 끝나야 다음 작업이 진행되는 방식

<br>

## 👩🏻‍💻 비동기(Asynchronous)
결과를 기다리는 동안 다른 작업도 진행할 수 있는 방식

<br>

## 👩🏻‍💻 블로킹(Blocking)
어떤 작업이 끝날 때까지 현재 흐름이 멈춰 있는 상태

<br>

## 👩🏻‍💻 Thread
작업을 실제로 실행하는 OS 수준의 실행 단위

<br>

## 👩🏻‍💻 RoomDB
SQLite를 객체 중심으로 쉽게 다루게 해주는 안드로이드 공식 DB 라이브러리

<br>

## 👩🏻‍💻 Primary Key
테이블에서 각 행을 고유하게 식별하는 기본 키 

<br>

## 👩🏻‍💻 Migration
DB 구조가 바뀌었을 때 버전을 올리고 안전하게 변경하는 작업

<br>
